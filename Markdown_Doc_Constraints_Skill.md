# Markdown 文档编写约束

## 适用范围

本约束适用于本目录下后续新增或更新的 Markdown 文档。

## 语言约束

- 默认使用中文（简体）编写全文。
- 除非用户明确要求英文或中英双语，否则不使用英文作为主语言。
- 代码、命令、寄存器名、信号名、路径名保持原文，不做翻译。
- 必要英文术语可保留，但需尽量配中文说明。

## 表达约束

- 标题与正文使用中文语义，避免中英文混杂造成歧义。
- 结论、风险、限制条件优先明确写出，避免模糊描述。
- 涉及数值、位宽、分辨率、单位时，必须写清楚单位与前提。

## 执行约定

- 从本文件创建后开始生效。
- 若后续用户给出新的文档语言要求，以用户最新要求为准。
